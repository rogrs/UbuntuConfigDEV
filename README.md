## Configurações

Edite o arquivo  abaixo:

    gedit ~/.bashrc

Adicione esta linha no arquivo ~/.bashrc

    source /home/$USER/profile


Para carregar as configurações  do profile execute o comando abaixo:

    source ~/.bashrc
